/body
Most tags of this website are made up of div tags and there are three main div tags. It is responsive using media queries and can be implemented in all screen sizes, including mobile.
@media screen and (max-width:1024px)
@media screen and (min-width: 1025px)

The first div tag is a cart section. When you click a cart on the left page of the homepage, the screen is pushed to the left and a cart about half the size of the screen appears. (.ajaxifyDrawer) When I am on the homepage Css does not seem to have any code to hide the cart, so it seems to have been implemented in JavaScript.

The second div tag is header and menu. The location of the navigation bar is different when it is full screen and when it is mobile. It is located on the left side when it is a full screen, but on mobile, you need to press the menu icon (which is .nav-mobile) to open a navigation. On a full screen, the header shares the same tag as the navigation bar, but in the mobile version, the css is different. It is located at the top of the screen.

The third div tag is the main homepage and footer. The homepage is made up of a main tag, and the designer used a grid to display four items in a row. Because of the media query, when the width decreases, the display changes to 2 items in a row. The image and product name of each grid element are composed of a tag and p tag. One interesting thing about footer is, instead of margins or padding, the designer used hr tags to create an empty space.
